W16-2

如何建立與推送fossil:

(fossil是老師為了github被封鎖的同學所建立的倉儲)

建立步驟:

1.到https://fossil-scm.org/home/uv/download.html下載windows64

2.去學校信箱，找scrum1(老師)寄給我們的一封信，裡面有密碼，點擊網址進入fossil測試倉儲(注意:最好點老師信件裡附的網址，若你使用他人給的網址，那麼你的密碼就算正確，fossil也不會讓你進入)

3.點擊倒數第二行的自訂密碼網址，點menu，點form，到達改密碼的地方，改成專屬於自己的密碼

4.接著打開可攜式系統，輸入:

cd tmp 

mdkir w16

cd w16

mkdir fossil

cd fossil

fossil clone https://409231xx@fossil.kmol.info/u/409231xx 409231xx.fossil (這裡主要是用來將fossil叫來，叫完後出現在fossil檔裡，問你新密碼後他會問你問題，什麼都說好就行，這裡特別要注意的是，空白鍵問題，沒有按空白鍵會失敗)

mkdir wd

cd wd

mkdir 409231xx

cd 409231xx

fossil open ./../../409231xx.fossil (這裡主要和fossil建立連結，連結完畢後出現在學號檔裡)

cd .. 

git clone --recurse- submodules https://github.com/409231xx/wcm2021.git(將github的資料複製一份，會問你帳號密碼，希望你沒有忘記自己的github帳密，注意:用github，不是fossil)

fossil add. (會跳出很長一串)

fossil commit "intial add"

5.推送完成即成功，可打fossil.kmol.info/u/409231xx/doc/trunk/content/index.html進入自己的fossil靜態頁面

開啟動態編輯加轉靜態推送步驟:

1.打開可攜式系統，輸入:

cd tmp

cd w16

cd fossil

cd wd

cd 409231xx

cms

(和當初進github一樣，轉靜態)

2.找wd檔裡的wcm2021裡的acp.bat，放進編輯器裡

3.接著把git都改成fossil，save

4.回到可攜式系統，打acp "xx"，就可以成功推送
